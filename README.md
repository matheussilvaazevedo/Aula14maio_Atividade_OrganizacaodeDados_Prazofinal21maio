# Atividade Prática: Estruturando Dados

## Objetivo
Transformar um conjunto de dados brutos em três representações computacionais distintas: lista, grafo e hierarquia.

## Tema escolhido
Alunos de uma turma.

## Estruturas apresentadas
- Lista: sequência de chamada dos alunos.
- Grafo: rede de amizades entre os alunos.
- Hierarquia: estrutura de liderança como representantes etc.

## Arquivos incluídos
- `Lista.md`: representação em lista.
- `Grafo.png`: representação visual do grafo.
- `Hierarquia.png`: representação visual da hierarquia.
